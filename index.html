Technical Experience & Applied Projects

Vision-Based Seismic Retrofit Identification

Repository ↗

Soft-story buildings are highly vulnerable to structural issues during earthquakes, often concentrating damage on the ground floor. To accelerate the identification of these at-risk structures, I developed a machine learning pipeline to prioritize San Francisco buildings for seismic retrofitting using Google Street View imagery, serving as a scalable screening tool to reduce manual urban inspections.

  • Engineered an automated building detection pipeline using Grounding DINO and RoboFlow.
  • Developed a custom classifier using PyTorch to process DINO embeddings, streamlining the architecture for rapid, high-volume screening.
  • Achieved 83.5% accuracy and 82.3% F1, outperforming vision-language models by ~14–17 points.
  • Reduced the number of required manual urban inspections by 51.3% while maintaining an 82.7% recall rate for at-risk structures.
  • Validated the model's adaptability by testing the pipeline against a dataset of residential houses in Los Angeles, proving broader scalability.
PyTorch, Grounding DINO, Torchvision, OpenCV
